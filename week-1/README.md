# Rule-Based AI Chatbot (week-1)

This is a simple rule-based AI chatbot implemented in Python for DecodeLabs internship (Week 1).

## Features
- Responds to greetings (hello, hi, hey, greetings)
- Asks if you are an intern and responds accordingly
- Answers questions like "how are you?", "what is your name?", "what can you do?", and "who created you?"
- Handles unknown input gracefully

## Usage
1. Open a terminal in the `week-1` directory.
2. Run the chatbot:

```bash
python chatbot.py
```

3. Type your messages and interact with the chatbot.
4. Type `quit`, `exit`, or `bye` to end the conversation.

## Example
```
You: hello
Chatbot: Hello there! Are you an intern at DecodeLabs? (yes/no)
You: yes
Chatbot: That's awesome! Good luck with Project 1!
```
